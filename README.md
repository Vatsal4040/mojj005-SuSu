# Manner Over Bladder

A situational logic game about public bathroom etiquette.

Each round presents a row of urinals.
Your goal is to choose the most polite option.

## Why this game exists
This game tests:
- Spatial reasoning
- Social intuition
- Common (and uncommon) etiquette rules

It’s intentionally opinionated and playful.

## How it works
- Each level shows occupied and empty urinals
- Select the best possible choice
- Some levels allow (or require) waiting instead
- Score increases with correct choices

There are a fixed number of levels.
The game ends when all scenarios are completed.

## Controls
- Click / tap on a urinal
- Or choose to wait when available

## Design principles
- Dark theme with high contrast
- Visual clarity over minimalism
- Humor without randomness
- Logic-first gameplay
